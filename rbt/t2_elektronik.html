2.4A
Mikropengawal (Microcontroller)
Microcontroller — control device in one chip
微控制器——单芯片控制装置
⌄
🟡 Definisi
Sebuah litar yang direka bentuk menggunakan komponen elektronik tertentu. Merupakan peranti kawalan dalam satu cip yang mengandungi CPU, RAM, ROM dan port I/O. Mengawal peranti elektronik berdasarkan pengaturcaraan yang telah dimuatkan.
Contoh: Kawalan pintu pagar automatik, mesin basuh.
🔵 Definition
A designed circuit using specific electronic components. A control device in one chip containing CPU, RAM, ROM and I/O ports. Controls electronic devices based on pre-loaded program.
Example: Automatic gate control, washing machine.
🟢 定义
使用特定电子元件设计的电路。包含CPU、RAM、ROM和I/O端口的单芯片控制装置。根据预装程序控制电子装置。
例子：自动门控制、洗衣机。
💡 Tip PLUS
Mikropengawal = komputer kecil yang menerima, memproses isyarat dan menghasilkan output.
Microcontroller = small computer that receives, processes signals and produces output.
微控制器=接收、处理信号并产生输出的小型计算机。

2.4A
Mikropemproses vs Mikropengawal
Microprocessor vs Microcontroller
微处理器与微控制器的比较
⌄
🟡 Mikropemproses
Peranti pemprosesan mikro dalam satu cip. Digunakan bersama cip-cip sokongan (Input, Output, RAM, ROM) yang berada di luar melalui System Bus. Berfungsi melakukan kawalan yang besar. Contoh: Kawalan komputer.
🔵 Microprocessor
Micro processing device in one chip. Used with supporting chips (Input, Output, RAM, ROM) outside via System Bus. Performs large-scale control. Example: Computer control.
🟢 微处理器
单芯片微处理装置。与外部支持芯片（输入、输出、RAM、ROM）通过系统总线配合使用。执行大规模控制。例子：计算机控制。

🟡 Ⓐ Bahagian Input
Menerima isyarat dan menghantar ke CPU.
Input Digital: menghasilkan isyarat digital.
Contoh: Suis tekan tutup, suis tekan buka, suis mikro, penderia pergerakan, penderia air.
Input Analog: menghasilkan isyarat analog.
Contoh: Penderia jarak inframerah, perintang peka cahaya (LDR), penderia suhu.
🔵 Ⓐ Input Section
Receives signals and sends to CPU.
Digital Input: produces digital signals.
Example: Push-close switch, push-open switch, micro switch, motion sensor, water sensor.
Analog Input: produces analog signals.
Example: Infrared distance sensor, LDR (light dependent resistor), temperature sensor.
🟢 Ⓐ 输入部分
接收信号并发送到CPU。
数字输入：产生数字信号。
例子：按压关闭开关、按压开启开关、微动开关、运动传感器、水传感器。
模拟输入：产生模拟信号。
例子：红外线距离传感器、光敏电阻（LDR）、温度传感器。
🟡 Ⓑ Bahagian Proses (CPU)
• CPU memproses berdasarkan atur cara dalam ROM (simpan kekal).
• RAM menyimpan data sementara (terpadam bila kuasa dimatikan).
• Isyarat dari input → CPU → menghasilkan isyarat kepada output.
🔵 Ⓑ Process Section (CPU)
• CPU processes based on program in ROM (permanent storage).
• RAM stores data temporarily (erased when power off).
• Signal from input → CPU → produces signal to output.
🟢 Ⓑ 处理部分（CPU）
• CPU根据ROM中的程序处理（永久存储）。
• RAM临时存储数据（断电时清除）。
• 输入信号 → CPU → 产生信号至输出。
🟡 Ⓒ Bahagian Output
Menerima isyarat daripada CPU dan menghantar kepada peranti output untuk dilaksanakan.
Contoh peranti output: Diod pemancar cahaya (LED), Motor arus terus (DC), Pembaz (Buzzer Piezo), Geganti (Relay).
🔵 Ⓒ Output Section
Receives signal from CPU and sends to output devices to execute.
Output device examples: LED, DC Motor, Buzzer (Piezo), Relay.
🟢 Ⓒ 输出部分
从CPU接收信号并发送到输出装置执行。
输出装置例子：发光二极管（LED）、直流电机、蜂鸣器、继电器。

2.4B
Pin-pin pada Papan Litar Mikropengawal
Pins on Microcontroller Circuit Board
微控制器电路板上的引脚
⌄
🟡 Reset
Disambungkan pada 5 VDC. Apabila disambungkan ke bumi (0 VDC), mikropengawal akan diset semula.
🔵 Reset
Connected at 5 VDC. When connected to ground (0 VDC), the microcontroller will be reset.
🟢 Reset（复位）
连接在5 VDC。当连接到地（0 VDC）时，微控制器将复位。
🟡 VDD
Pin bekalan kuasa pada mikropengawal. Disambungkan dengan sumber kuasa arus terus 5 V (5 VDC).
🔵 VDD
Power supply pin on microcontroller. Connected to DC power source 5 V (5 VDC).
🟢 VDD（电源引脚）
微控制器上的电源引脚，连接到直流电源5V（5 VDC）。
🟡 AIN1–AIN5 (Input Analog)
Input analog pada papan litar. Disambung dengan penderia analog (voltan 0–5 VDC). Nilai voltan analog ditukar ke digital oleh penukar analog-digital.
🔵 AIN1–AIN5 (Analog Input)
Analog input on circuit board. Connected to analog sensors (voltage 0–5 VDC). Analog voltage converted to digital by analog-to-digital converter.
🟢 AIN1–AIN5（模拟输入）
电路板上的模拟输入。连接到模拟传感器（电压0–5 VDC）。模拟电压由模数转换器转换为数字值。
🟡 DIN1–DIN4 (Input Digital)
Input digital pada papan litar. Pin telah disambungkan dengan perintang tetap 10 kΩ.
🔵 DIN1–DIN4 (Digital Input)
Digital input on circuit board. Pins already connected with fixed resistor 10 kΩ.
🟢 DIN1–DIN4（数字输入）
电路板上的数字输入，引脚已连接固定电阻10 kΩ。
🟡 DOUT1–DOUT6 (Output Digital)
Output digital pada papan litar. ON = 5 VDC (max 20 mA). OFF = 0 VDC (0 mA).
🔵 DOUT1–DOUT6 (Digital Output)
Digital output on circuit board. ON = 5 VDC (max 20 mA). OFF = 0 VDC (0 mA).
🟢 DOUT1–DOUT6（数字输出）
电路板上的数字输出。开 = 5 VDC（最大20 mA）。关 = 0 VDC（0 mA）。
🟡 VSS
Pin bumi mikropengawal. Pin disambung ke terminal negatif bekalan kuasa.
🔵 VSS
Microcontroller ground pin. Pin connected to negative terminal of power supply.
🟢 VSS（接地引脚）
微控制器接地引脚，连接到电源负极。

2 Jenis Lakaran Reka Bentuk Litar
2 Types of Circuit Design Drawings
两种电路设计图
⌄
🟡 A — Gambar Rajah Blok
Menerangkan secara ringkas cara sistem beroperasi. Setiap blok dilabel mengikut nama peranti. Format: Input → Proses → Output + Bekalan Kuasa.
🔵 A — Block Diagram
Briefly explains how the system operates. Each block labelled by device name. Format: Input → Process → Output + Power Supply.
🟢 A — 方框图
简要说明系统的运作方式。每个方框按设备名称标注。格式：输入→处理→输出+电源。
🟡 B — Lakaran Litar Skematik
Lakaran menggunakan simbol yang telah ditetapkan. Menunjukkan aliran kendalian bagi litar elektronik. Memudahkan pengguna memilih komponen untuk litar sebenar.
🔵 B — Schematic Circuit Drawing
Drawing using standardised symbols. Shows the control flow for electronic circuits. Helps users select components for the actual circuit.
🟢 B — 原理图
使用规定符号的图纸。显示电子线路的控制流程。帮助用户为实际电路选择元件。
🎯 Mesti Tahu / Must Know / 必须知道
Litar skematik memudahkan pengguna memilih komponen untuk litar sebenar.
Schematic circuit helps users select components for the actual circuit.
原理图帮助用户为实际电路选择元件。

Penilaian & Penambahbaikan Reka Bentuk
Evaluation & Design Improvement
评估与设计改进
⌄
🟡 Penilaian
Dilakukan bagi memastikan litar elektronik berfungsi seperti dikehendaki. Contoh: Apabila suis ditekan → LED berkelip-kelip.
🔵 Evaluation
Done to ensure the electronic circuit functions as required. Example: When switch pressed → LED blinks.
🟢 评估
确保电子线路按要求运作。例子：按下开关→LED闪烁。
🟡 Penambahbaikan
Dilakukan supaya reka bentuk dapat berfungsi dengan lebih baik untuk memenuhi kehendak pelanggan.
• Output: Tambah LED berlainan warna
• Input: Gantikan dengan penderia lain
🔵 Improvement
Done so design can function better to meet customer requirements.
• Output: Add LEDs in different colours
• Input: Replace with other sensors
🟢 改进
使设计能更好地运作以满足客户需求。
• 输出：添加不同颜色LED
• 输入：更换为其他传感器
🎯 Mesti Tahu / Must Know / 必须知道
Penambahbaikan reka bentuk litar elektronik dilakukan pada peranti input dan peranti output.

KawalanMudah/kecil / Simple / 简单Besar/kompleks / Complex / 复杂

3 Bahagian: INPUT (terima isyarat) → CPU/PROSES (ROM=kekal, RAM=sementara) → OUTPUT (laksana)
3 Sections: INPUT (receive signals) → CPU/PROCESS (ROM=permanent, RAM=temporary) → OUTPUT (execute)
三部分：输入（接收信号）→ CPU/处理（ROM=永久，RAM=临时）→ 输出（执行）
Input Digital: suis, penderia pergerakan, penderia air. Input Analog: LDR, penderia suhu, penderia jarak inframerah
Digital Input: switches, motion sensor, water sensor. Analog Input: LDR, temperature sensor, infrared distance sensor
数字输入：开关、运动传感器、水传感器。模拟输入：光敏电阻、温度传感器、红外线距离传感器
Output: LED, Motor DC, Pembaz (Buzzer), Geganti (Relay). Perintang 1kΩ = hadkan arus LED. Perintang 10kΩ = sambung pada suis input digital
Output: LED, DC Motor, Buzzer, Relay. 1kΩ resistor = limits LED current. 10kΩ resistor = connected to digital input switch
输出：LED、直流电机、蜂鸣器、继电器。1kΩ电阻=限制LED电流。10kΩ电阻=连接数字输入开关
Pin penting: VDD=5V bekalan | AIN=input analog (0–5V) | DIN=input digital (10kΩ) | DOUT=output digital (5V/20mA) | VSS=bumi
Key pins: VDD=5V supply | AIN=analog input (0–5V) | DIN=digital input (10kΩ) | DOUT=digital output (5V/20mA) | VSS=ground
重要引脚：VDD=5V电源 | AIN=模拟输入（0–5V）| DIN=数字输入（10kΩ）| DOUT=数字输出（5V/20mA）| VSS=接地
Arahan pengaturcaraan: If (Decision) + Dout/Timer (Controller I/O) + Goto + End If + ~Note (catatan) + Read_Controller
Program commands: If (Decision) + Dout/Timer (Controller I/O) + Goto + End If + ~Note (comment) + Read_Controller
编程指令：If（决策）+ Dout/Timer（控制器I/O）+ Goto + End If + ~Note（注释）+ Read_Controller
Bluetooth: PIN=1234 | Voltan=10–18V | LED modul berkelip = papan litar ON | Tekan SEARCH jika ID tidak muncul
Bluetooth: PIN=1234 | Voltage=10–18V | Module LED blinking = board ON | Press SEARCH if ID not shown
蓝牙：PIN=1234 | 电压=10–18V | 模块LED闪烁=电路板开启 | 若ID未显示按SEARCH
Pengujian = kekutuban betul + komponen betul + meter pelbagai. Penambahbaikan = output (LED warna lain) + input (ganti penderia)
Testing = correct polarity + correct components + multimeter. Improvement = output (different colour LED) + input (replace sensor)
测试=极性正确+元件正确+万用表。改进=输出（不同颜色LED）+输入（更换传感器）

Mikropengawal terdiri daripada bahagian input, proses dan output. Pengaturcaraan yang dimuatkan dalam ROM membolehkan CPU memproses isyarat input dan menghasilkan isyarat output untuk mengawal peranti elektronik.
A microcontroller consists of input, process and output sections. The program loaded in ROM enables the CPU to process input signals and produce output signals to control electronic devices.
微控制器由输入、处理和输出三部分组成。装载在ROM中的程序使CPU能够处理输入信号，并产生输出信号来控制电子装置。
