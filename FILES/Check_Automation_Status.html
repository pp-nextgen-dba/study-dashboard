Routine shows "Disabled"?

Solution: Go to routine page and click Enable button

Log file is empty?

Problem: Automation hasn't run yet, or no files detected

Solution: Upload a test HTML file and trigger manual run

Log shows validation error?

Problem: Something is wrong with the project

Solution: Check error message in log, fix issue, reupload

GitHub shows no auto-commits?

Problem: Files haven't been uploaded yet

Solution: Upload test file and wait for next cycle (or run manually)
